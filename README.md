# Docket 

Docket is your document-reading assistant.

Docket can assist you in highlighting and saving text, code, and images while you read documents on the web, especially for programming documents. It automatically recognizes the URL of the document site and organizes the excerpts for you.

## Getting Started

```bash
pnpm dev
```
